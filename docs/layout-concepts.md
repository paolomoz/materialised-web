## Block Patterns Summary

Each layout combines existing EDS block concepts with content-aware variations:

### Core Block Types Used
- **Hero** - Various styles (comparison, search, lifestyle, promotional)
- **Cards** - Product, recipe, feature, deal variations
- **Columns** - 2, 3, 4-up layouts for features and benefits
- **Carousel** - Product, recipe, accessory scrollers
- **Table** - Comparison, specs, ingredients
- **Accordion** - FAQ, features, troubleshooting
- **Grid** - Recipe cards, accessories, use cases

### Content Sources from RAG
- **Products**: 27 blenders with specs, features, pricing, reviews
- **Recipes**: 1,700+ with ingredients, instructions, nutrition, cook times
- **Support**: FAQs, troubleshooting, warranty info
- **Brand**: Heritage, value props, guarantees
